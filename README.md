 Carteirinha de Estudante

Projeto simples de front-end em HTML e back-end em JavaScript para gerar uma carteirinha de estudante dinâmica.O usuário preenche um formulário com seus dados e o JavaScript atualiza automaticamente a área da carteirinha.

🚀 Estrutura do Projeto

carteirinha/
│
├── index.html   # Página principal com formulário e área da carteirinha
└── script.js    # Lógica em JavaScript para atualizar os dados

🛠️ Tecnologias Utilizadas

HTML5 → Estrutura da página.

JavaScript → Manipulação dos dados e atualização dinâmica.

(Opcional) CSS → Para estilizar e deixar a carteirinha mais bonita.

📋 Funcionalidades

Formulário para entrada de dados:

*Nome
*Curso
*Matrícula
*Link da Foto
*Site da Instituição

Botão Gerar Carteirinha → Atualiza os dados na área da carteirinha.

Botão Limpar → Reseta o formulário.

Exibição dinâmica da carteirinha com:

*Nome do aluno
*Foto
*Curso
*Matrícula

Link para o site da instituição

▶️ Como Executar

Clone ou baixe este repositório.

Abra a pasta carteirinha no VS Code.

Instale a extensão Live Server (opcional, para recarregar automaticamente).

Abra o arquivo index.html no navegador:

Clique com o botão direito → Open with Live Server

ou simplesmente dê dois cliques no arquivo index.html.

Preencha os campos e clique em Gerar Carteirinha.

📷 Exemplo de uso

No campo Link da Foto, insira uma URL de imagem, por exemplo:

https://www.w3schools.com/howto/img_avatar.png
